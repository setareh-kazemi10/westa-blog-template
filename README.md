Javascript Dom Practice

A responsive blog template built with HTML, CSS, and Vanilla JavaScript.

## Features

- Dynamic DOM manipulation
- Responsive layout
- Blog homepage UI
- Sidebar
- Image gallery
- Footer section

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## Purpose

This project was built to practice JavaScript DOM manipulation by creating and managing page elements dynamically using DOM APIs.
